## Tech Stack

**Client:** HTML, CSS, JS

## Run Locally

Clone the project

```bash
  git clone https://github.com/ashu-sharma07/qr-code-app.git
```

Go to the project directory

```bash
  cd qr-code-app
```

```bash
  Open with Any Web Browser
```
